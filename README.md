# Paralelo-A-Uide
Fecha 21/12/2025

<div align="center">
  <h1> Juego del Ahorcado en Python</h1>
  <p>Proyecto academico desarrollado en Python</p>
</div>

<hr>

<div>
  <h2> Descripción</h2>
  <p>
    Este proyecto implementa el clásico <strong>juego del ahorcado</strong> utilizando el lenguaje
    de programación <strong>Python</strong>. El jugador debe adivinar una palabra secreta letra por letra
    antes de quedarse sin intentos.
  </p>
</div>

<hr>

<div>
  <h2> Características del juego</h2>
  <ul>
    <li>Selección aleatoria de palabras</li>
    <li>Validación de entrada del usuario</li>
    <li>Control de letras repetidas</li>
    <li>Límite máximo de intentos</li>
    <li>Interfaz clara por consola</li>
  </ul>
</div>

<hr>

<div>
  <h2> Tecnologías utilizadas</h2>
  <ul>
    <li><strong>Python 3</strong></li>
    <li>Librería estándar <code>random</code></li>
  </ul>
</div>

<hr>

<div>
  <h2> Cómo ejecutar el programa</h2>
  <ol>
    <li>Tener instalado Python 3 en el sistema</li>
    <li>Guardar el archivo como <code>ahorcado.py</code></li>
    <li>Ejecutar el programa desde la terminal:</li>
  </ol>

  <pre>
python ahorcado.py
  </pre>
</div>

<hr>

<div>
  <h2> Reglas del juego</h2>
  <ul>
    <li>Solo se permite ingresar una letra por turno</li>
    <li>No se aceptan números ni caracteres especiales</li>
    <li>El jugador tiene un máximo de 6 intentos</li>
    <li>Las letras usadas se muestran en pantalla</li>
  </ul>
</div>

<hr>

<div>
  <h2> Estructura general del código</h2>
  <ul>
    <li>Lista de palabras posibles</li>
    <li>Selección aleatoria de la palabra secreta</li>
    <li>Uso de listas para el progreso y letras usadas</li>
    <li>Bucle principal que controla el juego</li>
    <li>Condiciones de victoria y derrota</li>
  </ul>
</div>
